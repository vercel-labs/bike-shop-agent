# Spoke & Mirror torque reference

Fastener values in newton-meters (Nm). When in doubt, start at the low end and
check. Seeded into the sandbox at /workspace/torque-specs.md at session bootstrap.

| Fastener           | Torque (Nm) |
| ------------------ | ----------- |
| Disc rotor bolts   | 6           |
| Stem faceplate     | 5           |
| Seatpost clamp     | 5           |
| Crank arm (2-bolt) | 12          |
| Cassette lockring  | 40          |
| Pedal into crank   | 35          |
